# Laboratorio - Análisis de Algoritmos

Este repositorio contiene la implementación y análisis de algoritmos basados en recursividad y el paradigma de divide y vencerás.

## Contenido

- Implementación del problema del subarreglo máximo:
  - Fuerza bruta
  - Divide y vencerás (incluyendo max_crossing_subarray)
- Implementación de algoritmos de ordenamiento:
  - Merge Sort
  - Insertion Sort
- Análisis experimental de tiempos de ejecución para distintos tamaños de entrada
- Gráficas comparativas del comportamiento de los algoritmos

## Objetivo

Evaluar el desempeño de los algoritmos y comparar los resultados experimentales con su complejidad teórica.

## Resultados

Se observa que:

- Merge Sort presenta un crecimiento eficiente acorde a \(O(n \log n)\)
- Insertion Sort es eficiente en tamaños pequeños, pero su rendimiento disminuye rápidamente para entradas grandes debido a su complejidad \(O(n^2)\)

## Requisitos

- Python 3
- matplotlib
- seaborn

## Ejecución

Ejecutar los scripts en Python para reproducir los resultados y generar las gráficas comparativas.
